# 晶体管缺陷检测系统 — 测试文档

---

## 一、测试概述

### 1.1 文档目的

本文档详细描述晶体管缺陷检测系统的测试方案，包括测试计划、测试用例、测试执行记录和测试报告，确保系统功能的正确性、稳定性和性能达标。

### 1.2 测试范围

| 测试阶段 | 测试内容 |
|----------|----------|
| **单元测试** | 各功能模块的独立测试 |
| **集成测试** | 模块间接口和数据流转测试 |
| **系统测试** | 完整功能流程测试 |
| **性能测试** | 检测速度和资源消耗测试 |
| **验收测试** | 最终交付验收测试 |

### 1.3 测试环境

| 类别 | 配置 |
|------|------|
| **操作系统** | Windows 10 Professional |
| **CPU** | Intel Core i7-10700K |
| **GPU** | NVIDIA GeForce RTX 3060 |
| **内存** | 16GB DDR4 |
| **存储** | 500GB SSD |
| **软件** | HALCON 26.0 + HDevelop 26.0 |

### 1.4 测试数据

| 类别 | 数量 | 说明 |
|------|------|------|
| **训练集** | 200 张 | good（正常样本），含数据增强 |
| **测试集** | 100 张 | good 60 张 + 各类缺陷各 10 张 |
| **验证集** | 10 张 | good 样本，用于模型评估 |

### 9.3 测试指标计算公式

**准确率（Accuracy）**：
```
准确率 = (TP + TN) / (TP + TN + FP + FN)
```

**精确率（Precision）**：
```
精确率 = TP / (TP + FP)
```

**召回率（Recall）**：
```
召回率 = TP / (TP + FN)
```

**F1 分数**：
```
F1 = 2 × 精确率 × 召回率 / (精确率 + 召回率)
```

**IoU（交并比）**：
```
IoU = |预测区域 ∩ 真实区域| / |预测区域 ∪ 真实区域|
```
