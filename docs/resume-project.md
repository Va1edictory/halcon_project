# 项目经历：基于深度学习的晶体管缺陷检测系统

## 项目概述

基于 MVTec HALCON 深度学习平台，设计并实现了一套晶体管外观缺陷自动检测系统。该系统利用预训练的 GC Anomaly Detection 模型，实现对晶体管引脚、外壳等区域的异常检测与缺陷定位，支持单张调试和批量检测两种工作模式。

## 技术栈

- **视觉平台**：MVTec HALCON 26.05
- **深度学习**：HALCON DLT（Deep Learning Tool）— GC Anomaly Detection
- **数据集**：MVTec Anomaly Detection Dataset（晶体管类，4 种缺陷类型）
- **语言**：HDevelop（HALCON 原生脚本语言）

## 个人职责

- 独立完成从模型加载、图像预处理、异常推理、缺陷定位到结果可视化的全流程设计与开发
- 针对 HALCON DLT 模型输入要求（256×256，3 通道，值域 [-127, 128]，full_domain），实现手动图像预处理流水线（通道转换、尺寸缩放、值域映射、域处理）
- 基于异常热力图的阈值化与连通域分析，实现缺陷区域提取与矩形框标注，并针对多尺寸图像完成 256×256 异常图 → 原图尺寸的仿射坐标映射
- 设计双窗口交互式可视化方案：左窗展示原图，右窗展示 ORIGINAL 图像叠加红框标注 + 居中 OK/NG 判定 + 异常分数，契合工业质检场景
- 实现批量检测模块，支持文件夹遍历、进度反馈、按 OK/NG 分类保存结果图像、检测汇总统计，具备单张异常容错机制

## 项目成果

- 系统可准确判断晶体管是否存在缺陷并自动标注异常区域位置
- 支持交互式单张检测与离线批量检测两种工作模式
- 批量模式下自动保存分类结果，便于后续人工复核
